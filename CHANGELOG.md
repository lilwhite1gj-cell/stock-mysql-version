# 更新日志 (Changelog)

所有重要变更均会记录在此文件中。

## [1.1.0] - 2026-07-22

### 新增
- 🏷️ 版本号与更新日志系统，支持前端页面查看版本信息
- 🔄 GitHub Actions 自动部署工作流（推送到 main 分支自动同步至 Hostinger FTP）
- 🔒 服务端安全加固：CORS 收敛、请求限流、安全响应头
- 🛡️ 前端 XSS 防护：输入转义、上传文件类型校验

### 修复
- 🐛 修复删除流水记录时"撤销失败"无具体错误原因的问题，现在会显示服务端返回的详细错误信息
- 🐛 修复负库存校验删除入库记录时前端只显示通用提示的问题

### 变更
- ⬆️ cloudinary 依赖从 v1.41.3 升级至 v2.7.0（修复 CVE-2025-12613 任意参数注入漏洞）
- ⬆️ express 升级至 v5.2.1
- ⬆️ multer 升级至 v2.2.0

## [1.0.0] - 2026-07-10

### 核心功能
- 📦 库存台账管理：产品 CRUD、分类、SKU 货号、规格材质
- 📊 仪表盘看板：库存总览、收发统计、趋势图表
- 🔄 收发明细：入库/出库流水记录、图片凭证、撤销与库存回滚
- 🏭 工厂档案：供应商信息管理
- 👥 客户档案：客户信息与联系方式管理
- 🔐 用户认证：注册/登录、JWT Token、角色权限（admin/staff）
- 👑 账号管理：管理员可增删用户、调整角色权限
- 🏷️ 品类维护：自定义产品分类
- ⚠️ 库存预警：低库存自动提醒（阈值 10）
- 💱 汇率转换：CNY/USD 双币种支持，实时汇率查询
- 📤 数据导出：库存 CSV 导出、流水 CSV 导出
- 💾 数据备份：JSON 全量备份导出/导入恢复
- 🖼️ 图片上传：Cloudinary 云存储集成
- 📱 响应式设计：桌面端侧边栏 + 移动端底部导航
- 🗄️ MySQL 数据库：生产环境 MySQL + 开发环境 JSON 文件双模式
